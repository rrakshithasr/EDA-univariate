# 📊 Univariate Exploratory Data Analysis (EDA) on Online Shopping Orders

## 📌 Project Overview

This project demonstrates **Univariate Exploratory Data Analysis (EDA)** using Python. The analysis focuses on understanding one variable at a time through descriptive statistics, data visualization, and outlier detection.

The dataset contains information about online shopping orders, including customer details, product categories, prices, payment methods, and ratings.

---

## 🎯 Objectives

* Perform basic data exploration.
* Analyze numerical and categorical variables.
* Calculate descriptive statistics.
* Detect outliers using the IQR method.
* Visualize data using different charts.
* Interpret insights from the analysis.

---

## 📂 Dataset Features

| Column           | Description                  |
| ---------------- | ---------------------------- |
| Order_ID         | Unique order identifier      |
| Customer_Age     | Age of the customer          |
| Gender           | Customer gender              |
| Product_Category | Product category             |
| Quantity         | Number of products purchased |
| Price            | Total order price            |
| Payment_Method   | Payment method used          |
| Rating           | Customer rating              |
